# Jaranow Marketing Website

## ✅ Project Status: COMPLETED

This project includes complete landing pages and pricing for the Jaranow service lines:

1. **Home** - `/` - Overview of all Jaranow services
2. **Carwash by Jaranow** - `/carwash` - Fixed-price car wash in Gwarinpa, Abuja
3. **Jaranow Laundry Service** - `/laundry` - Premium subscription-based laundry service (legacy `/wash` redirects here)
4. **Pricing** - `/pricing` - Unified pricing page for all services
   - Supports URL parameters: `/pricing?service=carwash` or `/pricing?service=wash`
   - `/pricing?service=delivery` is kept as a legacy redirect to the car wash tab
